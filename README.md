# SONIQ-INVESTMENT
## What is ReelCash?
**ReelCash** is an online task-reward and VIP investment platform designed to allow users to earn income by subscribing to investment tiers, completing daily assigned tasks, and inviting new members through a multi-level referral system.
It functions as an all-in-one web platform where financial management, task distribution, and user permissions operate seamlessly in real time.
## Key Operating Modules
### 1. VIP Investment Tiers
The platform relies on a tiered membership model. Users purchase specific VIP packages to unlock earning potential.
 * **Investment Returns:** Higher-level packages offer higher daily returns and unlock access to premium tasks.
 * **Daily Earnings:** Users earn passive returns based on their active package level.
### 2. Daily Task Engine
To receive daily returns or additional task earnings, users interact with assigned daily activities.
 * **Scheduled Tasks:** Tasks are released daily based on server schedules.
 * **Media Verification:** Tasks often involve viewing or interacting with media content (videos, ads, or posts) uploaded by platform admins.
### 3. Multi-Tier Referral Program
ReelCash uses a multi-level commission system to encourage user acquisition:
 * **Level 1 (Direct):** Earn a commission percentage when a direct invite registers or buys a package.
 * **Level 2 & 3 (Indirect):** Earn smaller passive percentages from users invited by your referrals.
### 4. Financial Ledger (Deposits & Withdrawals)
 * **Deposits:** Users submit deposit requests via supported payment methods (such as local mobile money or crypto) along with payment receipts.
 * **Withdrawals:** Earnings can be withdrawn from either the task earnings balance or the referral balance, subject to platform minimums and admin review.
## User Roles & Hierarchy
The platform operates under a strict three-tier role system to keep operations secure and organized:
| Role | Responsibilities & Capabilities |
|---|---|
| **User** | Registers, deposits funds, buys VIP packages, completes daily tasks, tracks earnings, invites friends, and requests withdrawals. |
| **Admin** | Monitors user activity, approves or rejects manual deposit/withdrawal requests, and monitors general platform performance. |
| **Superadmin** | Holds full operational control—manages global settings (min/max deposit, commission rates), creates/edits tasks, manages administrative permissions, and deactivates accounts if necessary. |
## How the User Flow Works
 1. **Sign Up:** A new user creates an account (optionally using a sponsor’s referral link).
 2. **Fund Account:** The user submits a deposit request, which an admin verifies and approves.
 3. **Select VIP Package:** The user invests their account balance into a VIP package to unlock daily task limits.
 4. **Complete Tasks & Earn:** The user logs in daily, completes assigned media tasks, and collects task rewards + daily package returns.
 5. **Refer & Cash Out:** The user earns additional passive bonuses when their invites join, and can withdraw earnings once the minimum payout threshold is met.
